# 使用指南：

arduino的esp32版本为3.0.0

内存卡数据的config文件夹中的`config.txt`文件`hostName`对应我们的域名前缀，修改这里即可修改后台网址。

~~~
autoWifinum=1
wifiConnectTime=500
hostName=esp32
FirstWebis=1
~~~
